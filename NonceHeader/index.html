ECMAScript ImportMaps and Content Security Policy: Nonce (HTTP Header)

This page uses `nonce` values for its Content Security Policy, defined as `HTTP Header` tag, to authorize the scripts to run.

Content-Security-Policy:
  default-src
    'self';
  script-src
    'nonce-NqIr5ziC'
    'nonce-RXRHhu5O';

The `lib.js` is loaded by the `script.js` file without further `nonce` values.
